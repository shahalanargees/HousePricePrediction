# 🏠 House Price Prediction

A Machine Learning web application that predicts house prices based on user inputs. Built using **Python**, **Scikit-learn**, and **Streamlit**.

## Features

* Predict house prices instantly
* Interactive and user-friendly interface
* Real-time predictions

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
